# Kinetic Gain Embedded — tie-back

## What this repo demonstrates

This repo demonstrates the **exception routing** primitive — shipment exceptions tied to carrier rules, SLA pressure, resolution blockers, and operator-safe escalation.

## How Kinetic Gain Embedded extends it

Kinetic Gain Embedded is security-first embedded analytics for B2B SaaS.
It extends this primitive to logistics-aware and SLA-aware in-product analytics for fulfillment, support, and operations workflows, wrapped in:

- signed JWT authentication
- semantic-layer row-level security (RLS)
- read-only data connectors
- AST SQL validation
- append-only audit events
